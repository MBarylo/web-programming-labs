# Task Manager API (NestJS + PostgreSQL)

## 📌 Опис

Це REST API застосунок для управління задачами, реалізований на NestJS з використанням PostgreSQL та TypeORM.  
Дані зберігаються у базі даних, підтримуються зв’язки між задачами та тегами (Many-to-Many).

---

## ⚙️ Технології

- NestJS
- TypeORM
- PostgreSQL
- class-validator
- @nestjs/config

---
